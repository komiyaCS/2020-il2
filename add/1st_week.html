2020年度1Q　第1週

補足：キーボード

touch typing, タッチ・タイピング：　教育用計算機システムの Mac OS X には タイピスト Typist という練習ソフトが インストールされているので、練習したい人は遊んでみてください
Mac OS X と Windows などで Command/Control などのキー配列が微妙に異なります
Shift JIS キーボード（日本語） と US Keyboard などで記号配列が結構異なります
アルファベットの位置は同じです

補足：教育用電子計算機システム（演習室）と 個人所有のパソコン (PC) 端末

マシン（見た目）は同じですが、教育用電子計算機システムではどの端末にアクセスしても自分の情報が得られるようになっています
NFS (network file system) による個人ファイル管理です（容量 10GB）
NFS に保存していないファイル・データはログオフ・システム終了すると全部消えます
ソフト・アプリの個人設定は保存されます
使用できるようになったら、自分のパソコンとファイルのやり取りをは メール添付 や クラウド・ストレージ を利用しましょう
USB ストレージ は忘れものが多く、非推奨です（演習室での忘れ物）
